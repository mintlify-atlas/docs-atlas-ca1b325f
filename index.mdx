title: "DOSBox Staging: Modern DOS Emulation for Everyone"
description: "DOSBox Staging runs MS-DOS games and software on modern Windows, macOS, and Linux with improved audio accuracy, better graphics, and sensible defaults."
---

DOSBox Staging lets you run MS-DOS games and software on modern Windows, macOS, and Linux computers. It's a drop-in replacement for older DOSBox versions — your existing configurations continue to work while you gain access to improved audio accuracy, better graphics rendering, per-game configuration, and a growing feature set maintained by an active development community.
